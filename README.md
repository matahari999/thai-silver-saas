# SilverCare Thailand

แพลตฟอร์มจัดการดูแลผู้สูงอายุระดับพรีเมียม / Premium Elderly Care Management Platform

A multi-tenant SaaS platform for Thailand's premium nursing homes, elderly care centers, and wellness clinics. Manages patient records, daily care tracking, room/bed management, scheduling, billing with PromptPay QR, and family notifications via Line Messaging API.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | Next.js 14 (React) + TypeScript |
| **Styling** | CSS Custom Properties (dark/light ready) |
| **PWA** | next-pwa + Web App Manifest |
| **Database** | PostgreSQL 16 via Supabase |
| **Auth** | Supabase Auth + JWT + RLS |
| **API** | Next.js Route Handlers (REST) |
| **i18n** | Custom JSON-based (en / th) |
| **QR** | EMVCo-compliant PromptPay QR |
| **Notifications** | Line Messaging API + Line Notify |
| **Mobile Packaging** | Bubblewrap (PWA → Android AAB) |
| **CI/CD** | GitHub Actions (planned) |

## Setup

### Prerequisites

- Node.js 18+ (LTS recommended)
- npm 9+
- PowerShell 7+ (Windows) or bash (macOS/Linux)
- Supabase project (cloud or local)

### Quick Start

```powershell
# 1. Clone and enter project
cd D:\Workspace\thai-silver-saas

# 2. Install dependencies
npm install

# 3. Run full setup (generates icons, checks tools, creates .env)
npm run setup

# 4. Update .env.local with your credentials
#    - NEXT_PUBLIC_SUPABASE_URL
#    - NEXT_PUBLIC_SUPABASE_ANON_KEY
#    - LINE_CHANNEL_ACCESS_TOKEN (optional)

# 5. Apply database migrations (via Supabase dashboard SQL editor)
#    Open supabase/migrations/001_initial_schema.sql
#    Open supabase/migrations/002_additional_entities.sql
#    Copy-paste into Supabase SQL Editor and run

# 6. Start development server
npm run dev
```

## Key Features

### Round 1 — Core Operations
- **Resident Management**: Thai/English names, guardian info, bed assignment, medical records
- **Daily Care Records**: Meal, medication, bathroom, bathing, mobility, mood, vitals
- **Staff Accounts**: Role-based (ADMIN, MANAGER, STAFF, FAMILY) with row-level security

### Round 2 — Guardian & Admin Value
- **Family Reports**: Auto-generated daily/weekly/monthly summaries
- **Notifications**: Line Messaging API for billing alerts and incident reports
- **Billing**: Invoice with line items, PromptPay QR generation, payment recording

### Round 3 — Thai Localization & Expansion
- **Room/Bed Management**: Visual grid with status tracking (vacant, occupied, maintenance)
- **Admin Dashboard**: KPIs (occupancy, care completion, revenue)
- **Incident Tracking**: Falls, hospital visits, emergencies with guardian notification
- **Multi-language**: Full en/th support throughout UI and database

## Database Schema

The platform uses 14+ tables with strict multi-tenant isolation via PostgreSQL Row-Level Security:

- `tenants` — Nursing homes / facilities
- `profiles` — Users with role-based access (admin, manager, staff, family)
- `patients` — Residents with dual-language names + guardian info
- `rooms` / `beds` — Facility layout with status tracking
- `daily_care_records` — Per-resident, per-shift care documentation
- `incidents` — Event/accident reporting with severity levels
- `appointments` — Medical appointment scheduling
- `invoices` / `invoice_items` — Billing with line-item breakdown
- `payment_records` — Payment tracking (PromptPay, cash, bank transfer)
- `notification_log` — Email/Line/SMS notification history
- `translation_resources` — Dynamic multi-language content
- `line_notifications` — Line-specific notification records

## PromptPay QR

Generate EMVCo-compliant QR codes for Thai national ID (13-digit) or mobile number:

```typescript
import { generatePromptPayQR } from '@/lib/promptpay';

const qrData = generatePromptPayQR({
  targetId: '0812345678',    // Mobile or National ID
  amount: 1500.00,           // Amount in THB
});
```

## Line Notifications

Send alerts via Line Messaging API:

```typescript
import { sendBillingAlert } from '@/lib/line';

await sendBillingAlert(
  channelAccessToken,
  lineUserId,
  'สมชาย ใจดี',
  1500.00,
  '2026-08-15',
  'https://app.silvercare.com/invoice/INV-2026-00001',
  'th'
);
```
